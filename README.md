<h1>🧩ECR Project</h1>

<h3>프로젝트 소개</h3>
<ul>
  <li>포털 사이트 구축과 사용자의 편의성 증대</li>
  <li>한 화면에서 다양한 데이터를 볼 수 있도록 통합정보시스템 구축</li>
  <li>오픈소스 및 오픈 API를 활용한 플랫폼 구성</li>
  <li>통합 예약 시스템으로 빠르고 보기 쉽게 예약 가능</li>
</ul>
